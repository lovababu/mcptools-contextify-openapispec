

## Run chromadb as docker container
````bash
    $docker run -d -v ./chroma-data:/data --name chromadb -p 8000:8000 -e CHROMA_API_IMPL=rest -e CHROMA_SERVER_HOST=0.0.0.0 -e CHROMA_SERVER_PORT=8000 chromadb/chromadb:latest
````

## Test spec-vectorizer
````bash
    # Install dependencies
    $pip install -r spec-vectorizer/requirements_test.txt
    $PYTHONPATH=spec-vectorizer pytest spec-vectorizer/tests -s
````
